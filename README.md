*===LabGenius de Genomics===*:
Avec la participation de MPOHO Paulina, TOKOBANZA Exaucé et SINDA Victoire

LabGenius est une plateforme interactive de bio-informatique conçue pour les chercheurs et étudiants. 
Elle permet de simuler des manipulations génétiques complexes directement depuis un navigateur, avec un retour visuel en temps réel.

*==Fonctionnalités==*:

*Séquenceur de GénomeVisualisation interactive : Rendu dynamique des bases azotées (A, T, G, C) avec code couleur.

*Analyse en temps réel : Calcul automatique du taux de GC et décompte des bases.

*Transcription & Traduction : Passage instantané de l'ADN à l'ARNm et aux chaînes d'acides aminés (Phe, Met, Lys...).

*Simulateur de Mutations : Génération de mutations ponctuelles aléatoires avec historique complet des modifications.

*Bibliothèque GénomiqueBase de référence : Accès à des séquences célèbres (GFP, Résistance à l'Ampicilline).

*Persistance locale : Censer Sauvegardez vos propres créations dans le localStorage de votre navigateur.

*Favoris : Marquez les séquences importantes pour un accès rapide.

*Recherche intelligente : Filtrage dynamique par nom ou description.

*Technologies Utilisées HTML5 / PHP Structure de l'application et architecture modulaire. CSS3 (Variables & Flexbox) Interface futuriste (Neon Dark Mode) et responsive. JavaScript (ES6) Logique bio-informatique, et normalement un stockage local.

Configuration serveur avec le logiciel laragon  puisque le projet utilise des fichiers PHP pour les inclusions (header.php, sidebar.php).

*===Comment utiliser LabGenius===*:

L’application Labgenius a été développée conformément aux orientations générales du cahier des charges et assure le bon fonctionnement des fonctionnalités essentielles. Néanmoins, l’ensemble des exigences spécifiées n’a pas pu être entièrement respecté, principalement en raison de contraintes techniques et temporelles. Des optimisations et compléments restent envisageables dans le cadre d’évolutions futures

Ensuite prenez soint du code, malgrès les erreurs se code représente un grand travail pour nous 
vous devez utiliser un serveur local (XAMPP, laragon). 

*=Pour plus de précision liser la fiche pdf fournis=* 

Comment utiliser LabGenius ?Importer : Allez dans la Bibliothèque et cliquez sur "Charger" sur une séquence existante.

Analyser : Sur le Séquenceur, observez la structure des codons et la chaîne protéique résultante.

Modifier : Utilisez le bouton "Mutation Ponctuelle" pour simuler une erreur génétique.

Note aux développeurs : Ce projet a été conçu avec une approche "Front-end First" pour maximer la vitesse de production de l'app Web
